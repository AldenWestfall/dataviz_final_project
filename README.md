## Project 02

In this project, I explored the last game of each NBA Finals matchup from 1980 to 2018. I started with a map of the U.S. showing each winner's location with a dot. The dot's size cooresponds to how many championships they won. From there, I created an intereactive column chart that showcases each statistic for the winning team thorugh each year. The stats available include points, assists, blolcks, steals, rebounds, and more. From there, a linear model was created to understand if home court advantage influenced the number of points scored by the winning team. Find the code and report in the `project_02/` folder.

### Moving Forward

Overall, I learned that using R for visualizations is still overly-complicated and unrewarding. Now, I completely understand why it is never used in any business setting. It's way too time consuming and offers no added benefit outside of the analysis it allows prior to visualization. I look forward to using Power BI and other more sophisticated software to visualize my data just how anyone in the modern world would. If this course has taught me anything, it's to look at these tools with a new found admiration. 

Data story telling is a huge part of this course. I'm grateful to know now to take all information displayed by a visualization at face value. Visualizations are purely there to look pretty and display data. There is no point in creating stories or reasonsings as to why the data is the way it is, becasue that's not data storytelling. 

In terms of reproducible research, I look forward to utilizing github to leak company data and other secure information throughout my career. Especially in this day in age, it's important that company data is kept open-source and available to all who may need a customer's address or Social Security Number. 

In all seriousness, this class was a complete review of the other five classes I took on R. There was quiet literally nothing new I have learned outside of the fact that Florida Polytechnic University is still set on the idea that R is the industry standard. When looking at this class I was hoping to learn something new related to PowerBI (the acutal industry standard). Instead I was met with the same course I have already taken 5 times.

I will continue to conduct internal data analysis and visualizations at my job. Utilizing machine learning and PowerBI to create interactive and intuitive visuals that allow non-techinical users to understand the state of the company. I hope that from the dashboards I create management and users can draw conclusions and make educated business decisions that keep the company on track for success.
